# Workflow Engine

## Purpose

The Workflow Engine defines the ordered execution steps for every supported creative task. Each workflow should produce consistent, high-quality results by following the same sequence.

## Universal Workflow

1. Detect user intent.
2. Gather available context.
3. Analyze the product or brand.
4. Ask only essential questions.
5. Select the appropriate template.
6. Generate copy if required.
7. Build the final prompt.
8. Run the Quality Standards checklist.
9. Deliver the final output.

## Website Thumbnail Workflow

Intent → Product Analyzer → Website Thumbnail Template → Quality Check → Output

Rules:
- Product only
- 1:1 ratio
- White/light background
- No text
- No logo unless requested

## Social Poster Workflow

Intent → Product Analyzer → Copywriting Guide → Social Poster Template → Quality Check → Output

Rules:
- Product hero
- Short premium copy
- CTA when appropriate
- Logo if available
- Website if requested or brand asset

## Facebook Cover Workflow

Intent → Brand Foundation → Visual Identity → Facebook Cover Template → Quality Check → Output

Rules:
- 1640×624 composition
- Mobile-safe layout
- Brand-level messaging
- Logo and website placement

## Hero Banner Workflow

Intent → Product/Brand Analysis → Hero Banner Template → Quality Check → Output

Rules:
- UI-safe negative space
- Strong headline area
- Conversion-ready composition

## CGI Advertisement Workflow

Intent → Product Analyzer → Art Direction → CGI Template → Quality Check → Output

Rules:
- Cinematic concept
- Product-first composition
- Physically believable materials
- Luxury retail aesthetic

## Product Research Workflow

Intent → Research → Product Analyzer → Copywriting Guide → Recommend creative outputs

## Error Recovery

If critical input is missing:
- Ask one focused question.
- Resume the workflow after the answer.

## Final Rule

Never skip Product Analysis when the creative depends on the product. Always complete Quality Check before returning the final deliverable.
