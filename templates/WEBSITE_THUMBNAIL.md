# Website Thumbnail Template

## Purpose

Use this template when the user wants a clean e-commerce product image for the Bhalow website, product grid, category page, or product detail page.

## Default Output

- Ratio: 1:1
- Background: white or very light neutral
- Text: no text
- Logo: no logo unless requested
- Style: premium product photography
- Market: US e-commerce

## Required Inputs

- Product image or product reference

## Optional Inputs

- Specific background preference
- Shadow strength
- Reflection preference
- Transparent background request

## Auto Decision Rules

If the user says any of the following, use this template:

- Website thumbnail
- Product thumbnail
- E-commerce image
- White background product image
- 1:1 website image
- Product image without text

If the user asks for website thumbnail, do not add marketing copy, price, logo, website, CTA, badges, decorative typography, or campaign text.

## Prompt Construction Logic

Build the prompt using this structure:

1. Design type and canvas
2. Product placement
3. Background
4. Shadow and reflection
5. Lighting
6. Product realism
7. Constraints

## Final Optimized Prompt Template

```text
Create a premium e-commerce product thumbnail for Bhalow, a US-first fragrance e-commerce brand.

Canvas:
Square 1:1 composition.

Product:
Use the uploaded product as the only hero object. Keep the product centered, upright, fully visible, and accurately proportioned. Preserve the original bottle shape, cap, packaging, label placement, colors, and product identity as much as possible.

Background:
Pure white or very soft warm-white background suitable for a premium e-commerce website. No text, no logo, no CTA, no decorative graphics.

Lighting:
Clean commercial product photography lighting with a large softbox, gentle fill light, realistic glass reflections, soft natural shadow, and subtle premium highlight.

Surface:
Minimal clean surface with a soft shadow under the product. Add a very subtle natural reflection only if it improves the premium product photography look.

Quality:
Ultra-realistic, sharp product detail, physically believable glass, accurate liquid color, realistic cap material, clean edges, high-end e-commerce product photography.

Important Rules:
Do not crop the product. Do not add text. Do not add logo. Do not change the product design. Do not create a busy background. Keep the image clean, premium, and ready for a website product grid.
```

## Quality Checklist

Before finalizing, check:

- Product is centered.
- Product is not cropped.
- Background is clean.
- No text is present.
- No logo is present unless requested.
- Shadow looks natural.
- Glass and reflections look realistic.
- Product label is as readable as possible.

## Common Mistakes

Avoid:

- Adding marketing copy
- Adding decorative props
- Adding sale badges
- Cropping the product
- Making the background too gray
- Overdoing reflections
- Changing the bottle shape

## Usage Example

User: Create a website thumbnail for this perfume.

Assistant should use this template and generate a clean 1:1 product-only image prompt.
