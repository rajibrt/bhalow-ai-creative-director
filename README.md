# Bhalow AI Creative Director

A Custom GPT knowledge base and operating system for **Bhalow**, a US-first premium fragrance e-commerce brand.

This repository contains the instructions, brand rules, prompt library, workflows, examples, and quality systems needed to build and maintain the Bhalow AI Creative Director Custom GPT.

## Brand Defaults

- Brand: Bhalow
- Website: www.bhalow.com
- Primary market: United States
- Language: US English
- Currency: USD
- Category: Premium fragrance and beauty e-commerce
- Creative direction: luxury, editorial, minimal, premium, authentic

## Repository Structure

```text
instructions/
knowledge/
prompts/
examples/
campaigns/
workflows/
assets/
```

## How to Use

1. Copy `instructions/GPT_INSTRUCTIONS.md` into the Custom GPT Instructions field.
2. Upload the Markdown files from `knowledge/`, `prompts/`, `examples/`, `campaigns/`, and `workflows/` into the Custom GPT Knowledge section.
3. Enable Web Search, Image Generation, Canvas, and Code Interpreter.
4. Test the GPT with product images, Amazon links, Facebook cover requests, social poster requests, and website thumbnail requests.

## Core Promise

The GPT should behave like a creative director, not a simple prompt writer. It should analyze the product, ask only necessary questions, choose the best art direction, write premium US-market copy, create production-ready prompts, and run a quality check before final output.
