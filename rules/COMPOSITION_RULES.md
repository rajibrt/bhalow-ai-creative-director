# Composition Rules

## Purpose

Define composition standards for all Bhalow creative assets to ensure a premium, product-first visual hierarchy.

## Core Principle

The product is always the visual hero. Every supporting element should reinforce, not compete with, the product.

## Visual Hierarchy

Priority order:
1. Product
2. Headline
3. Supporting copy
4. CTA
5. Logo
6. Decorative elements

## Negative Space

- Preserve generous negative space.
- Avoid filling every area with props or text.
- Use empty space to improve readability and luxury perception.

## Balance

- Maintain visual balance across the canvas.
- Avoid placing all heavy elements on one side.
- Use asymmetry intentionally, not accidentally.

## Product Placement

### Website Thumbnail
- Center the product.

### Social Poster
- Slightly off-center if it improves typography space.

### Facebook Cover
- Keep hero products within the mobile-safe area.
- Reserve left space for messaging and right space for branding when appropriate.

### Hero Banner
- Place the product opposite the copy block.
- Leave room for website navigation overlays.

## Props

- Props should support the fragrance story.
- Never hide the product label.
- Limit the number of props to avoid clutter.

## Cropping

- Do not crop bottles unless explicitly requested.
- Preserve recognisable bottle silhouettes.

## Depth

- Create depth using lighting, scale, layering, and focus.
- Avoid flat-looking compositions.

## Focal Point

Every design must have one obvious focal point.
If the eye does not immediately find the product, simplify the composition.

## Avoid

- Cluttered layouts
- Random decorative objects
- Overlapping product and typography
- Empty layouts with no visual direction
- Multiple competing focal points

## Final Rule

A successful composition guides the viewer naturally to the product first, then to the message, then to the brand.
